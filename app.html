About AcaDNA

AcaDNA is an end-to-end machine learning project designed to analyze academic behaviors and identify student archetypes. Built as a Master's project, it combines clustering, classification, and explainable AI to provide personalized academic insights.

🧬 K-Means Clustering

Identifies 5 distinct student archetypes with 87% silhouette score

📊 XGBoost + SHAP

Explains cluster characteristics with interpretable features

🎯 KNN Classification

Predicts program alignment with 87% accuracy

🤖 Interactive Coach

AI-powered chatbot for personalized academic guidance

Tech Stack: Python · Scikit-learn · XGBoost · SHAP · HTML/CSS/JS · Three.js
